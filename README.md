## prod mode
1. run `make prod-up`
2. visit [api](http://localhost:8080/swagger/index.html) 

## dev mode
1. run `make dev-up`
3. visit [api](http://localhost:3000/swagger/index.html) 

## environments
### logger
- level
   - debug *(dev default)*
   - info *(prod default)*
   - warn 
   - error
   - fatal
