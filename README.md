## production mode
1. run `make prod`
2. visit [api](http://localhost:8080/swagger/index.html) 

## dev mode
1. run `make dev`
3. visit [api](http://localhost:3000/swagger/index.html) 

## environments
### logger
- level
   - debug *(dev default)*
   - info
   - warn *(prod default)*
   - error
   - fatal
